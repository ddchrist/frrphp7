Here is a brief explanation on the process to send me (Thales) IRFs using the girafe tool.
Using the girafe tool (http://10.17.43.200 on the administrative network, or the IRF icon on the small screen of CWPs of the TTF):
- Create an IRF with required information.
- Set Status to ToThales and Responsible to TATM
- Send me an email to notify me that there is a new IRF to analyze.
Once the analysis is done, and the IRF is agreed to be closed (following a de-briefing)
I will update the Girafe database with the answer to the IRF, the PCR number if there is one, and change the Status to FromThales and the owner back to NAV.
Please note that once an IRF has been set to ToThales and TATM, all the new changes made by NAVIAIR to the IRF inside the girafe tool will be discarded until I set it back to FromThales NAV (send an email instead to add inputs to the IRF).
